# How to run

1. Build the app: npm run build
2. Run the app: npm start

# How to update swagger docs
1. Run: npm run swagger
*Note that the swagger docs are automatically updated every time <pre>npm start</pre> is run.*

# How to run tests
<dl>
  <dt>Option 1: </dt>
  <dl>

  </dl>

  <dt>Option 2: </dt>
